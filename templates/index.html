如何创建模板
×
1. 模板文件格式：
  仅支持 .doc 或 .docx 格式的Word文档。

2. 占位符说明：
  模板中需包含以下占位符，系统会自动替换：
  • {{name}}：学生姓名
  • {{student_id}}：学号
  • {{class_name}}：班级
  • {{project_name}}：课程名称
  • {{instructor}}：导师
  • {{finish_date}}：完成日期
  • {{report_body}}：报告正文（正文模板必需）
3. 示例：
尊敬的{{instructor}}老师：

我是{{class_name}}班的{{name}}（学号：{{student_id}}），本次实训课程为《{{project_name}}》。

【正文内容】
{{report_body}}

完成日期：{{finish_date}}
4. 注意事项：
- 占位符必须用英文大括号包裹，如 {{name}}
- 正文模板必须包含 {{report_body}} 占位符，否则无法插入AI生成内容
- 可根据实际需要增删占位符，未填写的字段会自动留空

系统简介

本系统支持AI自动生成实训报告，支持多模板、资料上传、智能插图、格式自动修正等功能。

  • 注册/登录后可上传资料和模板
  • 支持多种资料格式和图片
  • AI自动插图，报告格式自动规范

使用流程

  1. 选择或上传Word模板（封面/正文）
  2. 上传资料文件（docx，md,支持图片）
  3. 填写基本信息，设置生成参数
  4. 点击"生成报告"，等待AI生成
  5. 下载Word报告，图片和格式均已自动处理

常见问题

  • 选择默认模板即可
  • 生成模式的区别：融合模式下，AI会根据所有资料融合为一个内容，区分模式下，AI会根据资料的类型，将资料分开，分别生成不同的内容
  • 高级prompt功能：在高级设置中，可以输入高级prompt，AI会根据prompt生成报告，如你想要的风格，要求等
  • 早期阶段，项目不稳定，更多问题请联系管理员

推荐使用方法

  1. 选择默认模板
  2. 选择区分模式（这样才会按照每个项目，逐段落的生成报告）
  3. 资料文档的内容不能太少，否则在多轮补全时，AI会自动发挥
  4. 设计要求及之后的四个空不写，系统会根据你的文档内容自动生成，写的话尽量不要写太多，会造成格式上的不美观(待修复，处理用户输入特别多要求设计等的提取情况)
