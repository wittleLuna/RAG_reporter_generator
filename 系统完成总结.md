## 🛠️ 技术栈

### 后端技术
- **FastAPI**: 现代化Web框架
- **SQLAlchemy**: ORM数据库操作
- **SQLite**: 轻量级数据库
- **Passlib**: 密码加密
- **Python-Jose**: JWT令牌
- **Uvicorn**: ASGI服务器

### 前端技术
- **HTML5**: 语义化标记
- **CSS3**: 现代化样式
- **JavaScript**: 交互功能
- **Font Awesome**: 图标库

### AI服务
- **OpenAI API**: GPT模型
- **千问API**: 通义千问模型
- **ChromaDB**: 向量数据库
- **Sentence Transformers**: 文本嵌入

### 文档处理
- **python-docx**: Word文档处理
- **PyPDF2**: PDF文档处理
- **Pillow**: 图像处理

## 🚀 部署方式

### 开发环境
```bash
# 1. 安装依赖
pip install -r requirements.txt

# 2. 配置环境变量
# 编辑 .env 文件

# 3. 启动服务
python start.py
```

### 生产环境
```bash
# 1. 运行部署脚本
python deploy.py

# 2. 配置Nginx (可选)
# 3. 启动服务
sudo systemctl start rag-system
```
